# laravel-docker-base
A starting point for laravel development using docker

### Prerequisites
* [Node](https://nodejs.org/en/) - Node w/ Package Manager
* [Docker](https://docs.docker.com/) - Image container 

### Get Started

To setup your new laravel project, run:
```
$ npm install
$ npm run start
```
